# Compendus Documentation

Compendus is a self-hosted personal digital library manager. Upload,
organize, and read books across web and iOS with automatic metadata,
full-text search, and reading progress tracking.

## Features

- **Multi-Format Support** — Read and manage PDF, EPUB, MOBI, CBR, CBZ ebooks, M4B/MP3/M4A audiobooks, and comic archives.
- **Automatic Metadata** — Fetch book metadata, covers, and descriptions from Google Books and Open Library automatically.
- **Reading Progress** — Track reading progress, create highlights with notes, bookmark pages, and log reading sessions.
- **iOS & Web** — Native iOS app with custom EPUB/PDF reader engine plus a full-featured responsive web interface.
- **Full-Text Search** — Search across titles, authors, descriptions, and optionally within book content itself.
- **Collections & Tags** — Organize your library with hierarchical collections, color-coded tags, and series tracking.

## Tech Stack

React 19, Vite, Hono, SQLite, Tailwind CSS, Drizzle ORM, SwiftUI, SwiftData
